# TikTok Game

لعبة تفاعلية تعمل مع بث TikTok المباشر.

## المميزات
- تفاعل مباشر مع التعليقات
- نظام نقاط للإعجابات
- واجهة مستخدم سهلة وبسيطة

## التشغيل
1. قم بتثبيت الاعتمادات: `npm install`
2. شغل الخادم: `npm run dev`
3. افتح المتصفح على: `http://localhost:3000`
